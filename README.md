## General info
This is simple stream application basing on Twitch idea
	
## Technologies
Project is created with:
* React (CRA)
* Redux and Redux Thunk
* AJAX
* json-server
	
## Setup
To run this project, install it locally using npm:

```
$ cd streams\client\
$ npm install
$ npm start
```

And also exactly same steps for our json database: 

```
$ cd streams\api\
$ npm install
$ npm start
```
*Notice that you will have to open two separate terminal windows in order to this properly. Also make sure that you are running the above from main catalogue*
